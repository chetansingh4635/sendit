Installation Guide:

1. First install nodejs on system.
       https://nodejs.org/en/

2. Download MongoDB.
     https://www.mongodb.org/downloads#production

3. Install RoboMongo
   http://robomongo.org/
[You can connect mongodb through Robomongo through the specified IP and PORT number.] 

4. Go to project directory, Where the app .js is located.

5. Open terminal [Verfiy terminal pointing the app.js folder]. 

6. Run "npm install" command.   

7. To start node server please run "node start".
